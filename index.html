  • 4. The product of 2 positive integers is 936. Find the greater number if the integers are in the ratio 13 : 18.

  • Solution:

    => x*x = ((936/13) * 18) = 4 => x = 2.

  • 8. If A : B = 2 : 3, B : C = 2 : 4 and C : D = 2 : 5, then what is A : D equal to?

  • Solution:

  • Given, A/B = 2/3, B/C = 2/4, C/D = 2/5
    Therefore, A/D = A/B * B/C * C/D = 2/3 * 2/4 * 2/5 = 2/15
    A:D = 2:15
